-Botão de jogar deve levar o jogador para um quiz (pode ter um sistema onde o usuário escolhe qual quiz vai querer jogar)
-Botões de respostas de perguntas, o jogador pode escolher uma resposta e passará para a próxima questão
-Aba "Login" leva o usuário para a tela de login, que deverá ser feito usando o e-mail e senha do usuário caso ele já tenha uma conta
-Aba "Cadastro" para novos jogadores entrarem com seu nome completo, tipo de perfil, e-mail, nome de usuário, uma senha e confirmar a senha.
-Aba "Desempenho" mostra como os usuários se saíram no quiz com seus nomes e respectivos acertos, erros e o tempo para concluir o quiz
-Aba "Scores" mostra o nome dos usuários e seus respectivos pontos e o tempo para concluir o quiz
-Aba "Menu" leva para a tela inicial onde o jogador pode ter uma breve apresentação do jogo e créditos
-Aba "Criação de questões" onde o professor pode criar, editar, remover ou pesquisar quantas questões quiser, e atribuir os respectivos pontos que essas questões valem
-Aba onde o jogador escolhe se vai criar um quiz ou jogar
-Aba "Criar quiz" o jogador selecionará as questões que ele já criou para serem perguntas no seu quiz ou pode optar por pesquisar questões do próprio quiz

Requisitos não funcionais:

-Suportar uma quantidade x de jogadores
-Ser responsivo a comandos do jogador
-Mostrar os pontos ou desempenho do jogador assim que possível
